Using theme components

In the parlance of Jekyll, components are called includes and live in the _includes folder. They are just like shortcodes in WordPress in that they’re quick, reusable blocks.

Wax comes with a lot of includes that you can browse on Github here. The easiest way to get started using Wax includes is by seeing how they’re used in the Wax demo site and trying them out on your own.
